# WordMaster Android

基于艾宾浩斯遗忘曲线理论的 Android 背单词应用。

## 功能特点

### 📚 短文学习
- 支持通过图片导入短文内容
- 按短文分类学习单词
- 实时跟踪学习进度

### 🧠 艾宾浩斯遗忘曲线
- 智能安排复习时间
- 科学对抗遗忘
- 个性化复习计划

### 📷 图片识别添加单词
- 拍照识别英文单词
- 从相册选择图片
- 支持批量添加

### 📊 学习统计
- 掌握进度可视化
- 连续学习天数
- 每日学习目标

## 技术栈

- **开发语言**: Kotlin
- **UI框架**: Jetpack Compose
- **架构**: MVVM + Hilt 依赖注入
- **本地数据库**: Room
- **文字识别**: ML Kit
- **最低支持**: Android 8.0 (API 26)

## 如何运行

### 前置条件
- Android Studio Hedgehog (2023.1.1) 或更高版本
- JDK 17
- Android SDK 34

### 安装步骤

1. 打开 Android Studio
2. 选择 "Open an existing Android Studio project"
3. 选择 `WordMasterAndroid` 文件夹
4. 等待 Gradle 同步完成
5. 连接设备或启动模拟器
6. 点击运行按钮

## 使用指南

### 添加短文
1. 将包含短文图片的文件夹命名为"短文1"、"短文2"等
2. 将文件夹放入应用支持目录
3. 重启应用自动导入

### 学习单词
1. 从首页点击"开始学习"
2. 选择要学习的短文
3. 查看单词卡片，点击显示释义
4. 选择"认识"或"不认识"

### 复习单词
1. 根据艾宾浩斯曲线自动安排复习
2. 收到复习提醒后开始复习
3. 系统会根据答题情况调整复习间隔

### 添加新单词
1. 点击底部"添加"标签
2. 选择拍照或从相册选择图片
3. 系统自动识别英文单词
4. 选择要添加的单词并保存
