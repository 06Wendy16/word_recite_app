# WordMaster - 智能单词记忆助手

一款基于艾宾浩斯遗忘曲线理论的 iOS 背单词应用，帮助你科学地记忆英语单词。

## 功能特点

### 📚 短文学习
- 支持通过图片导入短文内容
- 按短文分类学习单词
- 实时跟踪学习进度

### 🧠 艾宾浩斯遗忘曲线
- 智能安排复习时间
- 科学对抗遗忘
- 个性化复习计划

### 📷 图片识别添加单词
- 拍照识别英文单词
- 从相册选择图片
- 支持批量添加

### 📊 学习统计
- 掌握进度可视化
- 连续学习天数
- 每日学习目标

## 技术栈

- **开发语言**: Swift 5.9+
- **UI框架**: SwiftUI
- **本地数据库**: SQLite.swift
- **文字识别**: Vision Framework
- **最低支持**: iOS 16.0

## 如何运行

### 前置条件
- Xcode 15.0+
- XcodeGen
- iOS 16.0+ 模拟器或真机

### 安装步骤

1. 克隆项目
```bash
git clone <repository-url>
cd 单词记忆app
```

2. 生成 Xcode 项目
```bash
xcodegen generate
```

3. 打开项目
```bash
open WordMaster.xcodeproj
```

4. 在 Xcode 中选择目标设备和运行

## 使用指南

### 添加短文
1. 将包含短文图片的文件夹命名为"短文1"、"短文2"等
2. 将文件夹放入应用支持目录
3. 重启应用自动导入

### 学习单词
1. 从首页点击"开始学习"
2. 选择要学习的短文
3. 查看单词卡片，点击显示释义
4. 选择"认识"或"不认识"

### 复习单词
1. 根据艾宾浩斯曲线自动安排复习
2. 收到复习提醒后开始复习
3. 系统会根据答题情况调整复习间隔

### 添加新单词
1. 点击底部"添加"标签
2. 选择拍照或从相册选择图片
3. 系统自动识别英文单词
4. 选择要添加的单词并保存
